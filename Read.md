# C++ with ChatGPT
chatgptを使ってC++を学習する．chatgptへの命令は  
## input:
```md
私は，最近c++の学習を始めました．そこで番号と共にc++の問題を作成してください．
以下に問題の設定を載せます．
・回答は載せないでください．私が問題に応じて，ソースコード(回答)を作成し，こちらに載せますので，その際は採点をお願いします．
・問題の内容はアルゴリズムに重点を置かず，c++のクラスを理解するための内容にしてください．
・問題は難易度ごとに作成してください．初級を5問，中級を10問，上級を5問作ってください．
```
このプロンプトの重要な点は，  
- 回答は載せさせない．通常だと回答込みで問題作成してくる．
- 問題の内容をしていないとほとんどアルゴリズム寄りの出題だった．
- 問題のレベルは細かく指定して良いのかも．．例えば，初級では基本的なクラスの作成の問題を作り，中級ではポリモーフィズムを意識した問題にしなさい的な．
